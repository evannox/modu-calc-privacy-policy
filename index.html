1. 개인정보 처리방침 안내

noxdesign inc.(이하 "회사")는 모두계산기(ModuCalc) 앱(이하 "서비스")을 제공함에 있어 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다.

본 방침은 서비스 이용 과정에서 수집되는 개인정보의 항목, 수집 목적, 보유 기간 등을 안내합니다.

2. 수집하는 개인정보 항목

SDK / 기능수집 항목수집 방식
Google AdMob광고 ID (GAID/IDFA), 기기 정보, IP 주소자동 수집
자체 분석앱 사용 이벤트, 기기 정보, OS 버전자동 수집
  • 로컬 저장 데이터 (Hive): 설정값, 계산 기록, 테마 선택 등 — 기기에만 저장되며 서버로 전송되지 않습니다.
  • 환율 API 호출: 환율 정보 조회를 위한 네트워크 요청만 발생하며, 개인정보는 수집하지 않습니다.

3. 개인정보 수집 및 이용 목적

  • 맞춤 광고 제공 (Google AdMob)
  • 서비스 개선 및 사용 통계 분석 (Firebase Analytics)
  • 앱 안정성 확보 및 오류 진단

4. 개인정보 보유 및 이용 기간

  • 광고 ID: Google 광고 정책에 따름
  • 분석 데이터: 수집일로부터 14개월
  • 로컬 데이터: 앱 삭제 시 자동 삭제

5. 개인정보의 제3자 제공

  • Google LLC (AdMob) — 광고 제공 목적
  • 그 외 제3자에게 개인정보를 제공하지 않습니다.

6. 개인정보 처리 위탁

현재 개인정보 처리를 외부에 위탁하지 않습니다.

7. 이용자의 권리 및 행사 방법

  • 광고 ID 재설정/비활성화: 기기의 설정 > 개인정보 > 광고에서 변경할 수 있습니다.
  • 앱 삭제: 앱 삭제 시 기기에 저장된 모든 로컬 데이터가 완전히 삭제됩니다.
  • 개인정보 삭제 요청: 수집된 개인정보의 삭제를 원하시면 아래 이메일로 요청해 주세요. 요청 접수 후 지체 없이 처리합니다.
